# Venta de la tienda
Análsis exploratorio de 4 tiendas para identificar tendencias de cual debería venderse para comenzar con otro emprendimiento, esto basandose en las ventas, el tipo de ventas, la recepción de los ususarios que tienen de los productos.

##Descripción 
Este proyecto utiliza Python y librerías que procesan nuestros datos de los datasets, y generar conclusiones al respecto.También incluye librerías para la representacion visual y sean más entendibles nuestras conclusiones

## Tecnologías/herramientas utilizadas
* Pytho (Google Colab)
* Pandas
* De Matplotlip su módulo pyplot
